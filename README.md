# DrakesSlimeMarket

Mercado económico de materiales Slimefun para DrakesCraft. Lee los IDs de todos los addons instalados, pero no vende nada hasta que un administrador lo aprueba explícitamente en `config.yml`.

## Política

- Solo materiales configurados con ID, precio y cantidad.
- Infinity, Supreme, herramientas, armas, armaduras, máquinas y generadores quedan bloqueados en profundidad.
- Usa Vault para cobrar y Slimefun para entregar el ítem real.
- El comando interno es `/sfmercado`; Odysseia debe abrirlo desde la categoría de materiales del único `/tienda`.
